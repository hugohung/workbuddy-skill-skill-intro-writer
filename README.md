# Skill 介绍页文案生成器

> WorkBuddy Skill — 为 WorkBuddy skill 生成介绍页文案，按照固定格式输出

## 功能特性

- 按照「项目名称 / 项目描述 / 目标 / 状态 / 项目分类」五个字段的固定格式生成文案
- 自动识别 skill 核心功能，输出简洁准确的项目描述
- 聚焦价值与提效点，生成具体可量化的目标说明
- 支持一键生成，直接在对话中使用

## 安装方式

### WorkBuddy 用户

1. 下载 [Release zip](../../releases/latest)
2. 在 WorkBuddy 技能管理 → 上传技能，选择 zip 文件

### 从源码安装

```bash
git clone https://github.com/hugohung/workbuddy-skill-skill-intro-writer.git ~/.workbuddy/skills/skill-intro-writer
```

## 使用方式

在 WorkBuddy 对话中直接说：
> "帮我给这个 skill 写个介绍页"
> "生成 skill 介绍页文案"
